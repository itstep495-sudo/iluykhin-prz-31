# iluykhin-prz-31
Система выделения и маскирования объектов на изображении
Цель:
Автоматическая обработка изображений и скрытие объектов.

Используемые технологии:

Технология - Назначение
OpenCV - Обработка изображений
Haar Cascade - Детекция лиц
NumPy - Работа с массивами
Matplotlib - Отображение

Алгоритм работы:

- Пользователь загружает изображение.
- OpenCV выполняет обработку изображения.
- С помощью детектора Haar Cascade находятся лица.
- Обнаруженные области автоматически закрашиваются.
- Результат выводится пользователю.

Практическое применение:

- скрытие персональных данных
- анонимизация лиц
- модерация изображений
- предварительная обработка данных для ML-систем.
